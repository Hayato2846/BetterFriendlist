**A modern, feature-rich replacement for WoW's default Friends frame with powerful organization tools, raid management, and enhanced social features.**

Press **O** to open BetterFriendlist (replaces default Friends frame).

## ‚ú® Core Features

### üìÅ Custom Friend Groups
- **Create unlimited custom groups** to organize your friends list
- **Drag & drop friends** between groups with intuitive visual feedback
- **Collapsible groups** to keep your list clean and manageable
- **Favorites group** for your closest friends (automatic Battle.net favorite sync)
- **No Group category** for ungrouped friends
- **Accordion mode** - open only one group at a time to save space
- **Color-coded groups** - assign custom colors to each group
- **Group management** - rename, delete, and reorder groups easily
- **Import from FriendGroups** - migrate your existing groups with one click

### üéØ Advanced Filtering & Sorting

**Quick Filters** (one-click access):
- **All Friends** - show everyone
- **Online Only** - hide offline friends
- **Offline Only** - show who's offline
- **WoW Only** - filter to WoW players only
- **Battle.net Only** - show only Battle.net friends
- **Hide AFK** - exclude AFK/DND friends
- **Retail Only** - filter out Classic/other versions

**Dual Sort System** (primary + secondary):

Sort your friends list with two-level sorting - choose a primary sort, then a secondary sort for fine-tuned ordering:
- **Status** - Online ‚Üí Offline
- **Name** - Alphabetical (A-Z)
- **Level** - High to low
- **Zone** - Alphabetical by location
- **Activity** - Recently active first (based on your interactions)
- **Game** - Same WoW version first
- **Faction** - Same faction first (Alliance/Horde)
- **Guild** - Guildmates first
- **Class** - By role (Tanks ‚Üí Healers ‚Üí DPS)
- **Realm** - Same realm first

### üîç Powerful Search
- **Real-time search** across names, characters, and notes
- **Instant filtering** as you type
- **Works with all friend types** (Battle.net and WoW friends)

## üîÑ Migrating from FriendGroups

BetterFriendlist includes a built-in migration tool:

1. **Keep FriendGroups enabled** during migration
2. Open BetterFriendlist (press **O**)
3. Click the **Settings** button (bottom tab)
4. Navigate to the **Advanced** tab
5. Click **"Migrate from FriendGroups"**
6. Choose migration option:
   - **Migrate & Clean Notes** - Imports groups and removes FriendGroups syntax from notes
   - **Migrate Only** - Imports groups and keeps notes unchanged
7. After successful migration, you can disable FriendGroups

## üí° Tips & Tricks

### Friend Management
- **Right-click friends** for context menu (Whisper, Invite, Note, etc.)
- **Shift+Drag** to add friend to multiple groups
- **Regular Drag** to move friend (removes from other groups)
- **Click a friend** to select them (enables Send Message button)
- **Double-click** to quickly whisper a friend

### Display Customization
- **Show Notes as Name** - Great for remembering real names or nicknames
- **Treat Mobile as Offline** - Keep your online list focused on actual players
- **Window Scale** - Make the window bigger for better readability or smaller for more screen space

### Group Organization
- **Create focused groups** - PvP, Raid Team, Guildies, IRL Friends, etc.
- **Use colors** to distinguish group types visually
- **Collapse groups** you don't need to see right now
- **Use Accordion mode** when you have many groups

### Search & Filter
- **Combine filters and search** for precise results
- **Use secondary sort** for fine-tuned ordering
- **Quick filters save time** - no need to scroll through offline friends

### Raid Management
- **Select multiple before dragging** for bulk moves
- **Drag onto players** to swap positions
- **Use group headers** as drop zones for moving to groups

### Notifications (Beta)
- **Whitelist important friends** - Raid leaders, guild officers, close friends
- **Blacklist spam accounts** - Friends who log in/out frequently
- **Set quiet hours** - No notifications during sleep hours
