# BetterFriendlist Releases

This folder contains packaged release builds of the addon.

**v1.2.0-dev** (Development Build)
- File: `BetterFriendlist-1.2.0-dev.zip`
- Date: November 10, 2025
- Size: ~143 KB

### What's Included

The release package contains:
- ✅ All core addon files (`.lua`, `.xml`, `.toc`)
- ✅ All modules (Modules/)
- ✅ UI components (UI/)
- ✅ Utilities (Utils/)
- ✅ Documentation (README, CHANGELOG, SETUP_ANLEITUNG)

### What's Excluded

Development files are NOT included:
- ❌ Internal documentation (docs/)
- ❌ Development plans (plans/)
- ❌ Reference materials (reference/, predecessor/)
- ❌ Git files (.git, .gitignore, .pkgmeta)
- ❌ Development markdown files (PHASE_*, IMPLEMENTATION_*, etc.)

## Installation

1. Extract `BetterFriendlist-1.2.0-dev.zip`
2. Copy the extracted folder to your WoW AddOns directory:
   ```
   World of Warcraft\_retail_\Interface\AddOns\
   ```
3. Launch WoW and enable the addon in the AddOns menu

## Release Process

Release ZIPs are:
- **NOT** tracked in Git (excluded via .gitignore)
- Built locally for testing
- Distributed via GitHub Releases for official versions
